# Etch Intelligence

Etch Intelligence is the AI assistant built into Etch. It provides chat-based guidance and assistance, and it can build directly in your site — generating code, creating content, and setting up the data structures behind your pages, all from a conversation.

It lives in its own panel in the builder. Once activated, the panel appears automatically, and you can show or hide it at any time like any other panel in Etch.

## Two ways to work

Etch Intelligence has two modes, which you switch between using the toggle at the top of the chat input:

- **[Build Mode](./build-mode.md)** — the default. It's action-oriented: describe what you want and it produces insertable code and makes changes directly in your site.
- **[Ask Mode](./ask-mode.md)** — a conversational assistant for questions about Etch, web development, and how to accomplish things.

## Getting started

1. **[Activate Etch Intelligence](./activating-ai.md)** — the feature is currently experimental and requires an OpenAI API key. This page walks you through getting a key and turning it on.
2. Open the panel in the builder and start in **Build Mode**, or switch to **Ask Mode** when you just want answers.

## What's next

OpenAI is the only supported provider currently, but we plan to expand the available LLM options in the future to include additional providers and models. We also plan to provide a more capable, trained model natively via subscription.
